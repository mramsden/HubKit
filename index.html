HubKit by Marcus Ramsden

HubKit is an Objective-C library for communicating with the Github API.

HubKit is an Objective-C library for communicating with the Github API through your iOS and Mac OS X applications. At some point it will have a better homepage as well :)

Dependencies

json-framework - http://stig.github.com/json-framework/
